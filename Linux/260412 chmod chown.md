# chmod

只有文件夹所属用户或root用户可以修改权限

## 基本语法
chmod [-R] 权限 文件或文件夹

- `-R`（大写）：后面跟文件夹名时，对文件夹内所有内容应用同样的操作

## 示例
chmod u=rwx,g=rx,o=x hello.txt

- `u`：user，所属用户
- `g`：group，所属用户组
- `o`：other，其他用户

# chown

修改文件或文件夹的所属用户或用户组

**仅root用户可用**

## 基本语法
chown [-R] [用户:用户组] 文件或文件夹

- 冒号用于区分是用户还是用户组，其中一个不改则空缺
