# IP地址

- `127.0.0.1`：指本机
- `0.0.0.0`：指本机或白名单中任意IP或其他特殊关系

# Hostname

## 查看主机名
Hostname：查看Linux的主机名

## 修改主机名（root权限下）
hostnamectl set-hostname 新名称

# hosts文件
hosts文件里可以设置域名

# ping命令
可以检查指定的网络服务器是否为可联通的状态
ping [-c+num] ip或主机名
-c表示检查次数 eg.-c 3检查3次 不提供时无限次检查

# wget命令
wget [-b] url
-b:可选 指后台下载，且把日志写入

# curl 命令
curl [-O] url
发起网络请求
-O（大写）：下载文件
