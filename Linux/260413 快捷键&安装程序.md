# 快捷键

## 控制类
- `ctrl + c`：强制停止 / 换行重新输入
- `ctrl + d`：退出账户的登录

## 历史命令类
- `history`：查看历史输入过的命令
- `! + 命令前缀`：执行历史记录中上一次匹配该前缀的命令
- `ctrl + r`：搜索历史命令（输入关键词匹配）
  - 按回车键执行匹配到的命令
  - 按左右方向键可获取该命令到当前输入行（便于修改后再执行）
  
  > 示例：按 `ctrl + r` 后输入 `chm`，系统会匹配到最近的 `chmod ...` 命令，
  > 按右键即可将该命令放到当前命令行中，可以继续编辑而不立即执行。

## 光标移动类
- `ctrl + a`：跳到命令开头
- `ctrl + e`：跳到命令结尾
- `ctrl + 键盘左键`：向左跳一个单词
- `ctrl + 键盘右键`：向右跳一个单词

## 清屏
- `ctrl + l` 或 `clear`：清屏

## 粘贴
- 鼠标右键：粘贴

# 安装程序（yum）

## 基本语法
Yum [-y] [install安装 / remove卸载 / search搜索] 软件名称

- `-y`：无需手动确认

**注意：yum需要root权限**
