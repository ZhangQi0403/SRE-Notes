# Linux 命令学习笔记：`ls`

---

## 01. `ls` 命令 (列出内容)

`ls` 命令用于列出指定路径下的文件和目录。

### 1. 基础用法
* **命令：** `ls`
* **说明：** 列出当前工作目录下的内容。
* **环境备注：** * 在终端中，“Home” 通常是 `/home/用户名` 的显示名称。
    * 在你的主目录下运行 `ls` 等同于运行 `ls /home/heartfelt2amor`。
    * **根目录 (`/`)：** 在图形界面文件管理器中通常对应“计算机”或“此电脑”。它包含了 `home` 目录等所有系统文件。

---

### 2. 命令选项 (参数)

| 选项 | 名称 | 中文描述 | 英文描述 |
| :--- | :--- | :--- | :--- |
| `-a` | `--all` | **列出所有内容**：包括以 `.` 开头的隐藏文件。 | 列出所有条目，包括隐藏文件。 |
| `-l` | `长格式` | **长格式显示**：以竖向列表展示，并包含权限、大小、修改时间等详细信息。 | 以长列表格式显示输出及额外详情。 |
| `-h` | `--human-readable` | **易读模式**：配合 `-l` 使用，将文件大小转换为 KB, MB, GB 等单位（无单位时默认为字节 Byte）。 | 与 `-l` 配合使用，以易于阅读的格式显示文件大小。 |

---

### 3. 组合应用与语法

在 Linux 中，多个选项可以按任何顺序组合。以下命令在功能上是完全相同的：

* `ls -l -a`
* `ls -la`
* `ls -al`

**语法模板：**
```bash
ls [-a -l -h] [目标路径]
```
